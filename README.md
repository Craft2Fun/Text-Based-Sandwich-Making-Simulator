# 🥪 Text - Based Sandwich - Making Simulator 🥪

Welcome to my **Text - Based Sandwich - Making Simulator**! This is a simple and fun desktop app built with Python and PySide6. It lets you simulate building your dream sandwich step-by-step. Pick your favorite ingredients and create your personal masterpiece!

## 🌟 Features

- **Interactive UI**: A clean and simple desktop interface powered by PySide6.
- **Step-by-Step Builder**: Guided selection through different categories (Bread, Protein, Veggies, Sauces).
- **Flexible Input**: Choose ingredients by clicking buttons or typing them in the text box.
- **Smart Validation**: The app checks if your typed choice is actually on the menu.
- **Order Summary**: View a complete list of your selected ingredients at the end.
- **Quick Restart**: Ready for another round? Start a new sandwich instantly.

## 🚀 Getting Started

To run the Sandwich Maker on your local machine, you just need Python and the PySide6 library.

1.  **Install Python**:
    Make sure you have Python 3.x installed. You can download it from [python.org](https://www.python.org/).

2.  **Install PySide6**:
    Open your terminal or command prompt and run:
    ```bash
    pip install PySide6
    ```

3.  **Download the Project**:
    Clone this repository or just download the `main.py` file:
    ```bash
    git clone https://github.com/Craft2Fun/Text-Based-Sandwich-Making-Simulator.git
    ```
